# 实验二：失控重复触发探索

**目的：** 找到每个模型失控重复的触发条件——什么任务、什么参数下模型会"停不下来"。

**实验设计：**
- 重复梯度（5/10/20/50/100 次）
- 自然触发（写修仙小说、列举 100 个成语、超低温说明文、出数学题）
- 三模型同一条件对比

**核心发现：**
- **Gemma 3 1B 在要求 50 次时爆发失控**（实际 282 次，5.6 倍超额），要求 100 次时 347 次
- 1B 写小说也自然触发了失控重复
- **e2b 和 9B 在所有条件下均未失控**——失败模式是"欠交"（主动停下）而非"超交"（停不下来）
- 9B 的欠交天花板约 54 次，e2b 约 72 次

**数据文件：**
- `trigger_explore_*.md` — 汇总报告
- `trigger_full_*.json` — 完整数据（含原始输出）
- `trigger_raw_*.json` — 精简数据
