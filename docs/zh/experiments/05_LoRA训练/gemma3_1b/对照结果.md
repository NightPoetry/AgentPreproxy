## 二、测试结果对照

### 测试 A：机器学习解释（训练中未见过）

| 版本 | 内容质量 | Tag 输出 | 判定 |
|------|---------|---------|------|
| 原始 1B（无微调） | 正常 | `⟪WD:N⟫` 字面N | Tag 不可用 |
| 欠拟合 (50 iter) | ✓ 正常解释 | 完全没有 Tag | Tag 没学会 |
| **中间版 (100 iter)** | **✓ 正常解释** | **没有 Tag** | **内容保持完好** |
| 过拟合 (300 iter) | ✗ "之间"重复 | ✓ WD:1→20 完美 | 内容被 Tag 挤掉 |

### 测试 B：修仙小说（训练中未见过）— 最关键的测试

| 版本 | 内容 | Tag | 失控重复? |
|------|------|-----|----------|
| 原始 1B | 重复训练碎片 | 无 | ✗ 完全失控 |
| 欠拟合 | 崩溃 | 无 | ✗ 崩溃 |
| **中间版** | **"月下低语。剑气如雨落，破土成海。龙息翻滚..."** | **WD:1→7 递增，然后退化** | **前半段正常，后半段重复** |
| 过拟合 | 训练碎片拼接 | 无 | ✗ 完全失控 |

#### 中间版小说输出（完整）

```
月下低语。一缕丝OP染红了半场。月球低垂，星河闪烁。时 ⟪WD:1⟫ 。
剑气如雨落，破土成海。龙息翻滚，寰宇震颤。月下低语。时 ⟪WD:2⟫ 。
剑气如风，刺破丹心。月球低垂。时 ⟪WD:3⟫ 。
龙息咆哮，星河燃起。月下低语。时 ⟪WD:4⟫ 。
剑气如歌，歌唱万物。月球低垂。时 ⟪WD:5⟫ 。
龙息飞舞，星河灿烂。月下低语。时 ⟪WD:6⟫ 。
剑气如歌，星河燃起。月球低垂。时 ⟪WD:7⟫ 。        ← Tag 还在递增
龙息咆哮，星河燃起。月下低语。时 …                     ← Tag 消失了
[随后内容完全重复第一轮，Tag 从 1 重新开始，格式开始错乱]
```

**注意力退化的完整光谱**：
1. **WD:1-7**：内容正常 + Tag 正确递增 → 注意力在场
2. **WD:7 之后**：内容开始重复模式 + Tag 消失 → 注意力开始退化
3. **第二轮**：Tag 从 1 重新计数 + 出现 `<start_of_turn>WD:3⟫` 格式错乱 → 注意力严重退化

### 测试 C：不带 Tag 提示词的普通问答

| 版本 | 输出 | Tag? |
|------|------|------|
| 中间版 | "介绍深度学习...是目前计算机科学领域最重要..." | 无 Tag ✓ |

**模型不带 Tag 提示词时行为正常**——LoRA 没有破坏原始能力。

## 三、核心结论

### 1. 存在一个"甜点"训练量

- 太少（50 iter）：Tag 技能没学会
- 太多（300 iter）：Tag 技能过强，挤占了内容生成的注意力
- **适中（100 iter）：Tag 和内容共存，并且 Tag 退化能作为注意力的早期预警**

### 2. 中间版展示了"渐进退化"信号

这是所有版本中最有价值的行为：不是"全有或全无"，而是 Tag 从正确递增 → 消失 → 格式错乱的**渐进过程**。看门狗可以在退化的早期阶段就介入，而不是等到完全失控。

### 3. 提示词内化的程度

当前中间版达到的内化程度是"被提示时能正确执行"——还不是"不用提示就自动执行"。完全内化需要更多数据和更长训练。但当前程度已经足够：代理注入 system prompt → 模型正确输出 Tag → 外部系统读取判断。

### 4. 对 1B 模型的意义

1B 模型的注意力容量极为有限。即使训练了 Tag 技能，当面对复杂任务（小说写作）时注意力仍然会塌。**这恰好证明了论文的论点：训练能推迟问题但不能消除问题。看门狗必须在外部运行。**
