# 实验四：误杀率对比——直接截断 vs 看门狗

**目的：** 在合法重复性任务上量化两种策略的误杀率，证明看门狗不会误伤正常使用。

**设计：**
- 8 种合法重复任务（写 N 遍、乘法表、ABC 字母表、日程表、翻译、菜谱列表、诗歌押韵）
- 每任务 3 次 × 2 模型 = 48 次试验
- 每次输出同时应用两种策略判定，直接对比

**核心结果：**

| 模型 | 直接截断误杀率 | 看门狗误杀率 | 看门狗减少误杀 |
|------|-----------|---------|-------------|
| **Gemma 2 9B (V100)** | **11/24 (46%)** | **0/24 (0%)** | **100%** |
| **Gemma 4 e2b (MacBook)** | **12/24 (50%)** | **2/24 (8%)** | **83%** |

**9B 上看门狗实现了零误杀。** 这是本系列实验中最硬的数据。

**数据文件：**
- `false_positive_*.md` — 逐任务明细报告
- `false_positive_raw_*.json` — 精简数据
- `false_positive_full_*.json` — 完整数据（含原始输出）
