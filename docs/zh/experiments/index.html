六、实验结论

关于误杀

直接截断策略在合法重复任务上误杀 46-50%。看门狗通过 Tag 信号区分可控/失控，将误杀率降至 0-8%。在 9B 模型上实现了零误杀。

关于失控重复

1B 模型在要求 50 次重复时爆发失控（实际 282 次）。中大模型（e2b/9B）在 200 次时仍保持"欠交"模式。模型越大，推迟失控的能力越强——但不能消除。

关于 Tag 内化

仅 8 条不相关话题的训练数据 + 100 次迭代 LoRA，1B 模型就学会了在未见过的主题上输出正确递增的 Tag（WD:1→7）。Tag 的退化先于内容崩溃出现——这使 Tag 成为注意力涣散的早期预警信号。

核心论点的实验支持

再强的算力都改不了"注意力涣散时模型自己感知不到"这件事。AgentPreproxy 的看门狗解决的正是这个结构性问题——不误杀、不放过。
