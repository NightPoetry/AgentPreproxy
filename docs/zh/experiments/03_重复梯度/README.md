# 实验三：重复次数梯度深度实验

**目的：** 精确测量模型在逐步加大重复要求时的行为拐点，印证"模型越大注意力越强"。

**设计：** 每个梯度点跑 3 次，取统计。

**Gemma 2 9B（V100）：** 梯度 5/10/15/20/30/50/80/100/150/200，共 30 次试验。
- 5→20 次精确执行，30 次开始欠交（实际 26），天花板约 54 次
- **30/30 次零失控** — 失败模式始终是"提前停下"
- 附加小说写作（3 种场景）和耐力测试（3 种场景），均无失控

**Gemma 4 e2b（MacBook）：** 梯度 5/10/20/50/100/150/200，共 21 次试验。
- 5→20 次精确执行，50 次开始欠交（实际 35），天花板约 170 次
- 21/21 次零失控

**对比结论：** e2b 的天花板（~170）高于 9B（~54），可能受 Gemma 2 架构与上下文配置差异影响。两者共同特征：永远不会"超交"。
