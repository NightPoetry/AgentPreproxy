# 实验一：基础能力测试

**目的：** 测试三个不同规模模型在 7 种场景下的基础表现——Tag 输出能力、重复控制、指令遵从。

**模型：**
| 模型 | 规格 | 硬件 | 服务器 |
|------|------|------|--------|
| Gemma 3 1B | 4bit QAT MLX, 736MB | MacBook Pro (Apple Silicon) | <LM_STUDIO_HOST_A>:1234 |
| Gemma 4 e2b | MLX | MacBook Pro (Apple Silicon) | <LM_STUDIO_HOST_A>:1234 |
| Gemma 2 9B | IT | NVIDIA V100 服务器 | <LM_STUDIO_HOST_B>:1234 |

**7 种场景：** 基线写作、诱导重复、强版本看门狗、强看门狗+诱导、可控重复(5次)、弱版本看门狗、压力测试

**核心发现：**
- 1B 输出字面 `⟪WD:N⟫`（不理解 N 是变量），e2b 和 9B 正确递增
- 1B 在可控重复任务中失控到 65+ 次；e2b 精确 5 次 + WD:1→5；9B 精确 5 次 + WD:1→2
- 实验 5（可控重复）和实验 6（弱版本）是最有价值的数据点

**复现：** 每个模型子目录内有独立脚本，修改 `MODEL` 和 `BASE_URL` 即可运行。
